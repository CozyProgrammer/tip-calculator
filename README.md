# Tip Calculator

Bill splitter that updates live as you type. Enter a bill, 
pick a tip percentage, add how many people — done.

## How to Run

Download the repo and open `index.html` in a browser.
No installs. No commands. Nothing.

git clone https://github.com/CozyProgrammer/tip-calculator.git
Then just open `index.html`.

Deployed version: [https://cozyprogrammer.github.io/tip-calculator/]

## What's in it

- Live results — no calculate button needed
- Preset tip buttons (10%, 15%, 20%) plus custom input
- Error messages that appear inline, not as popups
- Reset button
- Works on mobile

## Built with

HTML, CSS, vanilla JavaScript. No frameworks.
